# Waste to Gas (W2G)

W2G is a web platform for managing and incentivizing waste-to-gas conversion. Users can submit waste, track their impact, subscribe to gas plans, and view AI-powered insights (coming soon). The project features a clean, eco-friendly design and is built with HTML, CSS, and JavaScript.

## Features
- Waste submission and tracking
- User dashboard with incentives
- Subscription management
- Roadmap and contact info
- AI Assistant (coming soon)
